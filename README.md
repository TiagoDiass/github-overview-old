# Github Overvue

## To start the project you can follow these steps :arrow_heading_down:
```bash
# Clone the repository
$ git clone https://github.com/TiagoDiass/github-overvue.git

# Enter in the repository
$ cd github-overvue

# Install the dependencies
$ yarn install

# Run the app
$ yarn start
```
